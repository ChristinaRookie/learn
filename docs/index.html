Electric Book documentation

These docs explain how to set up the Electric Book template, how to edit book files, and how to output books. The template includes the docs, so that any Electric Book includes the docs for its own version (this is version 0.10.0).

These docs are a work in progress, and not everything is documented yet. You can contribute on GitHub, or let us know about problems by logging issues.
